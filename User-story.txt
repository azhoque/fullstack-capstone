Project description 
Ontrack is a web-application to that helps project teams to keep records of daily/weekly project status and next steps.

User stories 
As a user, I should be able to create a project
As a user, i should be able to save project
As a user, I should be able to edit project
As a user, I should be able to delete project
As a user, I should be able to enter a name for a new project
As a user, I should be able to enter task name for a project
As a user, I should be able to enter team members name 


As a user, I should be able to create a task
As a user, I should be able to save task
As a user, I should be able to edit task
As a user, I should be able to delete task
As a user, I should be able to enter description for a task
As a user, I should be able to choose a due date for a task
